# SRE System Monitoring Tool

This project is a Bash-based system monitoring tool.

## Features
- CPU usage monitoring
- Disk usage alert
- Service health check with auto-restart

## Files
- cpu_check.sh
- disk_check.sh
- service_check.sh
- monitor.sh

## How to Run
chmod +x *.sh
./monitor.sh
